## Vue 다운로드
### Vue를 다운로드하기 위해 Node.js 설치
```
https://nodejs.org/ko
```

### Node.js 설치 후 확인 과정
#### cmd
```
node -v
```

### workspace 폴더 생성

```
npm init vue
```

```
cd 생성한 폴더
```

### 생성한 Project의 terminal
```
npm install
npm run dev
```
